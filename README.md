# ERP (Enterprise Resource Planning)
WPF 기반 ERP 데스크톱 앱과 이메일 인증기능을 제공하는 프로젝트입니다.

## 기술 스택
- 언어: C#
- UI: WPF
- Backend: ASP.NET
- ORM: Entity Framework Core
- DB: PostgreSQL

## 주요 기능
- 로그인/권한 기반 메뉴 제어
- 회원가입 이메일 인증 (숫자+영어(대문자 + 소문자), 제한시간 3분)
- 품목/재고/사용자 관리
- 감사 로그

## 수정 예정 사항 
- Window App (.exe) 생성 예정
- 메뉴 탭 세분화 및 UI 생성 예정
